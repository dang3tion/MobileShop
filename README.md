ĐỒ ÁN MÔN HỌC - Môn Học : Lập trình web 
Thời gian hoàn thành : 12/2020

Front-End
- bootstrap 4
- java JSP

Back End
- Java servlet
- Java JDBC

Database : 
- Microsoft SQL server 2019 

API, plugin, add-on :
- facebook comment,
- Login with google,
- Google Recaptcha V2,
- Zalo chat widget,
- CKEditor,
- Google Drive API.

Mô hình, Design Parten :
- MVC, singleton, DTO.
M -> java beans, utility model.
V -> java JSP .
C -> java servlet, filter.
Singleton -> DAO, BO services.
